# Proiect SmarTest - Livrabil L6 (MinMax)

Acest proiect conține implementarea end-to-end pentru generarea și evaluarea problemelor MinMax cu tăieri Alpha-Beta.

## Tehnologii

* **Backend:** FastAPI (Python)
* **Frontend:** React.js
* **Bază de date:** PostgreSQL
* **Orchestrare:** Docker Compose

---

## 1. Rulare Proiect

Proiectul este complet containerizat. Singura cerință este să aveți Docker și Docker Compose instalate.
